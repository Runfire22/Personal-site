Using Font Awesome 5 icons on Next JS project
    1. Install dependencies
npm install @fortawesome/fontawesome-svg-core @fortawesome/free-solid-svg-icons @fortawesome/react-fontawesome

    2. Add the necessary code to _app.js
import "@fortawesome/fontawesome-svg-core/styles.css"; // import Font Awesome CSS
import { config } from "@fortawesome/fontawesome-svg-core";
config.autoAddCss = false; // Tell Font Awesome to skip adding the CSS automatically since it's being imported above

    3. Add code to page where you want to use it
import { FontAwesomeIcon } from "@fortawesome/react-fontawesome"; // Import the FontAwesomeIcon component
import { fa[Icon name] } from "@fortawesome/free-solid-svg-icons"; // import the icons you need

Place tag <FontAwesomeIcon icon={fa[Icon name]} where you want it.

You can see available icons in the following file:
/node_modules/@fortawesome/free-solid-svg-icons/index.d.ts

Or website: https://fontawesome.com/icons?d=gallery
